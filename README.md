O TourDB é um gerenciador de shows desenvolvido para facilitar o cadastro e o controle de informações relacionadas a bandas, membros, locais e shows. A aplicação permite criar, visualizar e associar esses elementos, possibilitando uma gestão completa de eventos musicais.

### FERRAMENTAS UTILIZADAS

**Frontend**

- React.js (Vite)

- React Router DOM

- Axios

- CSS Modules

- JavaScript

**Backend**

- Spring Boot (Java)

- H2 Database

- Spring Data JPA

- Maven 

### FUNCIONALIDADES PRINCIPAIS

- Login / Autenticação:
Usuário realiza login e recebe um token armazenado no localStorage.

- Cadastro de Bandas:
Permite registrar novas bandas vinculadas a um show existente.

- Cadastro de Membros:
Adiciona membros associados a uma banda específica.

- Cadastro de Shows:
Registra eventos (nome, data e local).

- Cadastro de Locais:
Mantém locais onde os shows acontecem.

- Listagem e Gerenciamento:
As páginas “Bandas” e “Membros” exibem os dados cadastrados via backend.
